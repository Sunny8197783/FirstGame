# Idle Pawn Shop: 지하경제 (FirstGame)

낮에는 전당포에서 물건을 감정·흥정하고, 밤에는 지하 격투장에서 베팅하거나 직접 싸우는 **3막 21일 경영 시뮬레이션 데모**.

## 실행

```bash
npm install
npm run dev        # 개발 서버 (http://localhost:5173)
npm run build      # 정적 빌드 → dist/
npm run simulate   # 헤드리스 회귀 시뮬레이션 (흥정·스파링)
```

Vite + 바닐라 JS ES 모듈 (프레임워크 없음). 원본 단일 파일 데모는 `legacy/demo_v1.html`에 보존.

## 구조 (Phase 1 모듈화)

```
src/config.js      모든 밸런스 수치 (데모 CONFIG 그대로)
src/data/          items · customers · fighters · rumors · events · story
src/systems/       haggle · appraisal · betting · sparring · market · economy · save
src/ui/            title · day · night · fight · result · hud · pixelart · rig · debug
src/audio/         Web Audio 삑삑이
sim/               헤드리스 회귀 시뮬 (npm run simulate)
```

이식 원칙: 데모의 함수를 **원문 그대로** 모듈로 분리(로직 무변경), 모듈 간 호출·인라인 onclick은 globalThis 등록으로 해석. 회귀 시뮬로 흥정 매입가율·결렬률·스파링 승률이 데모와 동일 범위임을 확인함.

## 엔드게임·콘텐츠 (Phase 3 신규)

- **🌆 시즌 모드**: 21일 캠페인 정상 클리어 시 해금되는 무한 모드. 6주(42일) 시즌마다 파이터 하위 2명 은퇴 → 신인 데뷔(랜덤 스탯·필살기), 전설 아이템 세트 3종(왕실 예물/전장 유물/밀수 보물)이 시즌별 로테이션, 시즌 마지막 밤은 그랜드 파이널. 기존 시스템만 재사용(새 시스템 없음).
- **🏚️ 가게 이전(프레스티지)**: 시즌 모드의 새벽 암시장에서 선택. 누적 1회당 아이템 가치 +50%(판돈·수익 동반 상승) ↔ 함정 힌트 +5%p·헛소문 +5%p. 위험과 보상을 함께 올리는 선택.
- **콘텐츠 증량**: 아이템 22→**36종**(전설 세트 12종 포함, 픽셀아트·검수 부위 완비), 손님 9→**14종**, 소문 문구 테마당 3→**10개**(총 60), 일일 이벤트를 **선택지형 10종**으로 개편(비용 지불 → 효과 반영, 통보형 폐지).
- **후일담**: 시즌 모드 아침에 김사장·회장·강 형사의 후일담 대사(`EPILOGUE_LINES`).
- **밸런스 패스**: `INSULT_RATIO` 0.65→0.68. `npm run balance`로 4개 플레이 스타일의 성사율·매입가율 측정 (무지성 +27% / 신중 +11% / 고수 +31% / 공격 결렬 93% — 스킬 보상·리스크 구조 확인).

## 하루 루프

**낮 (전당포)** — 손님 4~6명과 **밀당 흥정**: 손님이 요구가를 부르면(유형별 뻥튀기/투매 성향) 제시가로 밀당. 낮게 부를수록 요구가가 내려오지만 인내심(❤️)이 깎이고, 최저 수락가의 65% 미만은 모욕으로 박차고 나감. 부위별 실물 검수(1~5점)·구두 힌트(25% 함정)·픽셀아트 실물로 진짜 가치를 추정. 🎉 숨은 진품(5%, 되팔이 ×2)과 🚨 장물(수상한 유형 12%, 50% 경찰 압수) 변수. **진짜 가치는 저녁 정산에서 공개.**

**밤 (지하 격투장)** — 12명 파이터(팔다리 리그 + 기술 8종: 펀치/킥/플라잉니킥/슈퍼맨펀치/업어치기/발목후리기/바디프레스/급소공격). 하우스 배당은 스탯·전적만 반영, **소문은 플레이어만 아는 정보**. 소문 6테마(🩹🍶🌫️🔥⚡🎯) — 같은 아이콘이 겹치면 사실, +/− 충돌 시 하나는 거짓(추리 시스템). 경기 아크: 압승/접전/역전극(+진짜 소문 인용 **각성** 연출). 2경기 후 **직접 출전**(러시>견제>카운터 심리전, 상대 텔 읽기 — 60% 진짜/40% 스마트 페인트).

**새벽 (암시장)** — 빚 상환/대출, 도구 구매(정밀 저울·감정 렌즈·학회지·정보원·확장·리모델링·큰손 인맥), 파이터 후원 계약(승리 지분 +300G, 소문 전체 검증).

## 시뮬레이션 인바리언트 (Phase 5 신규)

헤드리스 봇으로 **게임의 핵심 설계 원칙을 자동 검증**한다 (`npm run simulate`).

| 봇 | 정책 | 검증하는 원칙 | 실측 |
|---|---|---|---|
| `BOT_IDLE` | 아무것도 안 함 | **방치 보상 = 0** | 21일 후 골드 변화 **0G**, 배드엔딩 100% |
| `BOT_RANDOM` | 무작위 제시·베팅 | 무작위는 하우스 마진에 마모 | 베팅 기대값 −, 1막 통과 15% |
| `BOT_SMART` | 최저가·소문 진위 활용 | **판단력 = 수익** | 진엔딩 92%, 완주 92% |

`npm run balance` → 흥정 4스타일 리포트 + 하루 루프 리포트(순자산 곡선·막 관문 통과율·엔딩 도달률·수익 출처).
밸런스 판독 기준: **방치 0 → 무작위 마모 → 정보 활용만 우상향**.
