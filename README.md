# Idle Pawn Shop: 지하경제 (FirstGame)

낮에는 전당포에서 물건을 감정·흥정하고, 밤에는 지하 격투장에서 베팅하거나 직접 싸우는 **3막 21일 경영 시뮬레이션 데모**.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다. 빌드 도구·외부 라이브러리 없음 (바닐라 JS + CSS 단일 파일).

## 게임 구조 (3막 캠페인)

| 막 | 기간 | 목표 |
|---|---|---|
| 1막 김사장의 빚 | 1~7일 | 빚 10,000G(이자 5%/일 복리)를 갚아라 — 8일 아침 강제 회수, 실패 시 배드엔딩 |
| 2막 회장의 시험 | 8~14일 | 순자산 40,000G 달성 — 실패 시 "회장의 감정사" 엔딩 |
| 3막 지하경제의 왕 | 15~21일 | 21일 밤 👑 그랜드 파이널(베팅 상한 100%) 후 최종 결산 — 순자산별 엔딩 3종 |

## 하루 루프

**낮 (전당포)** — 손님 4~6명과 **밀당 흥정**: 손님이 요구가를 부르면(유형별 뻥튀기/투매 성향) 제시가로 밀당. 낮게 부를수록 요구가가 내려오지만 인내심(❤️)이 깎이고, 최저 수락가의 65% 미만은 모욕으로 박차고 나감. 부위별 실물 검수(1~5점)·구두 힌트(25% 함정)·픽셀아트 실물로 진짜 가치를 추정. 🎉 숨은 진품(5%, 되팔이 ×2)과 🚨 장물(수상한 유형 12%, 50% 경찰 압수) 변수. **진짜 가치는 저녁 정산에서 공개.**

**밤 (지하 격투장)** — 12명 파이터(팔다리 리그 + 기술 8종: 펀치/킥/플라잉니킥/슈퍼맨펀치/업어치기/발목후리기/바디프레스/급소공격). 하우스 배당은 스탯·전적만 반영, **소문은 플레이어만 아는 정보**. 소문 6테마(🩹🍶🌫️🔥⚡🎯) — 같은 아이콘이 겹치면 사실, +/− 충돌 시 하나는 거짓(추리 시스템). 경기 아크: 압승/접전/역전극(+진짜 소문 인용 **각성** 연출). 2경기 후 **직접 출전**(러시>견제>카운터 심리전, 상대 텔 읽기 — 60% 진짜/40% 스마트 페인트).

**새벽 (암시장)** — 빚 상환/대출, 도구 구매(정밀 저울·감정 렌즈·학회지·정보원·확장·리모델링·큰손 인맥), 파이터 후원 계약(승리 지분 +300G, 소문 전체 검증).

## 코드 구조 (index.html 단일 파일)

- **CONFIG** (파일 상단): 모든 밸런스 수치 — 흥정(INSULT_RATIO/ACCEPT_GAP), 잭팟/장물, 빚/이자, 막 목표, 배당 공식, 소문 확률, 스파링 등
- **콘텐츠 데이터**: ITEMS 22종(픽셀아트 PIXEL_ART, 검수 부위 ITEM_PARTS), CUSTOMER_TYPES 9종(desp/ask/pat), FIGHTERS_DATA 12명, RUMOR_THEMES 6종, FIGHT_MOVES 8종, DAILY_EVENTS, UPGRADES, INTRO_BEATS/INTERLUDES
- **디버그**: 게임 중 `[D]` 키 — 진짜 가치·최저 수락가·소문 진위·실제 승률·EV 표시
- HUD "결과 화면 보기" 버튼으로 중간 결산 가능

## 검증 상태

- 헤드리스 시뮬레이션 통과: 흥정(신중 92% / 고수 66% 매입가율, 공격 시 결렬 12%), 스파링(막러시 -EV / 텔 읽기 +EV), 막 전환 5분기 전부 브라우저 실측
- 콘솔 에러 0
