## 1. 이 게임이 뭔가요?

**낮에는 전당포에서 물건을 감정·흥정하고, 밤에는 지하 격투장에서 베팅하거나 직접 싸우는 3막 21일 경영 시뮬레이션.** 단일 데모에서 시작해 정식 출시 품질로 끌어올리는 중입니다.

**핵심 재미 = "정보를 읽는 판단력 = 수익".** 방치하면 골드가 1G도 안 늘고, 모든 수익은 능동적 판단(흥정 밀당, 소문 추리, 텔 읽기)의 결과입니다.

### 하루 4박자 루프
| 시간 | 내용 |
|---|---|
| ☀️ 낮 | 손님이 **요구가**를 부르면 제시가로 밀당(인내심❤️·모욕 임계). 부위별 실물 검수(●1~5점)+구두 힌트(함정 섞임)로 진짜 가치 추정. 🎉진품(5%,×2)·🚨장물(압수) 변수. **진짜 가치는 저녁에 공개** |
| 🌆 저녁 | 매입품 되팔이 정산 — 낮의 판단이 여기서 채점됨 |
| 🌙 밤 | 파이터 베팅(하우스는 스탯·전적만 앎, **소문은 나만 아는 정보**). 소문 6테마 겹침=사실/충돌=거짓 추리. 경기 아크(압승/접전/역전+각성). 2경기 후 **직접 출전**(러시>견제>카운터 심리전, 텔 60%진짜/40%페인트) |
| 🌌 새벽 | 암시장: 빚 상환/대출, 도구 7종, 파이터 후원, (시즌 모드)가게 이전. **수동 저장** |

### 3막 스토리 + 시즌 모드
- **1막(1~7일)** 김사장의 빚(이자 5%/일 복리) → 8일 아침 강제 회수, 못 갚으면 배드엔딩
- **2막(8~14일)** 회장의 시험(순자산 40,000G) → 실패 시 엔딩
- **3막(15~21일)** 21일 밤 👑그랜드 파이널 → 순자산별 엔딩 3종
- **🌆 시즌 모드**(클리어 후 무한): 6주 시즌, 파이터 물갈이, 전설 세트 로테이션, 🏚️가게 이전 프레스티지

## 3. 시작하는 법

```bash
npm install
npm run dev        # 개발 서버 http://localhost:5173
npm run build      # 정적 빌드 → dist/
npm run simulate   # 회귀 시뮬(흥정·스파링) — 변경 후 반드시 통과 확인
npm run balance    # 흥정 밸런스 리포트(4가지 플레이 스타일)
```

게임 실행 후 **[D]키 = 디버그 패널**(진짜 가치·최저 수락가·소문 진위·EV 표시), HUD 우측 📈📊🏆⚙️ 아이콘으로 통계/확률/업적/설정 열람.


### ⚠️ 절대 원칙 — 보존 대상 (로직 변경 금지)
데모에서 검증된 시스템은 **동작을 바꾸면 개선이 아니라 버그**입니다:
- 3막 캠페인·엔딩 분기 / 밀당 흥정(INSULT_RATIO·ACCEPT_GAP) / 감정 정보구조(검수·함정·잭팟·장물)
- 소문 추리(겹침=사실/충돌=거짓) / 배당 공식 `(1/추정승률)×0.95` / 정보 비대칭(하우스는 소문 모름)
- 경기 아크·각성 / 스파링 심리전(텔 60/40) / 새벽 암시장 / **방치 보상=0**

### 이식 코드의 관례
- Phase 1에서 데모 함수를 **원문 그대로** 모듈로 분리 → 모듈 상단에 `globalThis` 등록으로 상호 호출·인라인 onclick 해석
- 신규/수정 코드는 `[Phase2]`, `[Phase3]` 주석으로 마킹 (이식 원문과 구분)
- **변경 후 `npm run simulate` 통과 필수** (흥정 매입가율·결렬률·스파링 승률이 기준 범위 유지되는지 회귀 검증)

## 7. Phase 4 착수 체크리스트 (다음 작업)
1. **사운드**: `src/audio/audio.js`의 단일 beep → 상황별 SFX 12종+(흥정 성사/결렬/모욕, 잭팟, 압수, 베팅 승/패, 각성, 드롭, 레벨업…) + Web Audio 신스 BGM 2트랙(낮 lo-fi/밤 신스). 첫 입력 후 시작, 음소거 상태에서도 완전 동작. **SETTINGS.sfxVol/bgmVol은 이미 존재** — 연결만 하면 됨.
2. **픽셀아트**: `PIXEL_ART` 팔레트 테마당 12색 이내로 정리, 손님 감정 변형 2~3종, 엔딩 분기별 SVG 일러스트.
3. **UI 시스템**: 색 토큰(낮 앰버/밤 네온을 시맨틱 변수로), 8px 스페이싱, 버튼/카드/모달/토스트/툴팁 통일, 골드 카운트업.
4. **접근성**: 승패·손익 색+아이콘 이중 코딩, WCAG AA 대비, 키보드 단축키.
5. **반응형**: 1280×720~1920×1080, 60fps.
